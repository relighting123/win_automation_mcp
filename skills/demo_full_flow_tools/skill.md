# Demo Full Flow Skill
이 스킬은 애플리케이션을 실행하고, 텍스트를 입력한 뒤, 단축키(Ctrl+S)를 누르고 특정 키워드를 클릭하여 종료하는 전체 흐름을 시연합니다.

### 지침
1. `{username}`은 사용자의 실제 성함이나 계정명으로 치환하세요.
2. `{confirm_keyword}`는 화면에 나타날 확인 버튼의 텍스트(예: "확인", "Yes", "OK")로 치환하세요.
3. 각 단계가 성공적으로 수행되는지 확인하며 진행하세요.

### 🛠 도구별 세부 주의사항
- **launch_application**: 이 앱은 실행 후 메인 화면이 뜨기까지 약 5초 정도 걸립니다. `wait_for_window` 인자를 반드시 `true`로 설정하세요.
- **type_app_text**: 한글 입력 시 자음/모음이 분리될 수 있으니 `interval`을 평소보다 조금 더 긴 `0.05` 정도로 설정하는 것이 안전합니다.
- **click_app_by_keyword**: "제출" 버튼이 두 개 있을 수 있습니다. 반드시 상단 메뉴에 있는 것이 아닌, 하단 팝업창 내의 버튼을 찾으라고 가이드하세요.
