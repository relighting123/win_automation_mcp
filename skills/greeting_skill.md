# Notepad Greeting Skill

This skill types a welcome message when an empty Notepad window is detected.

## Trigger
- Window Title: Contains "Untitled" or "제목 없음"
- UI Element: Document area is focused

## Actions
1. Type "Hello! I am your AI assistant. This action was triggered by a Skill."
2. Press "Enter"
3. Type "How can I help you today?"
