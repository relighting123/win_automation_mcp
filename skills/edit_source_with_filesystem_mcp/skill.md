# Edit Source with FileSystem MCP
이 스킬은 FileSystem MCP의 표준 도구를 사용해 파일을 수정합니다.

## 목적
- 파일에서 수정할 대상을 확인하고
- 필요한 최소 범위만 바꾼 뒤
- 변경 결과를 다시 읽어 검증합니다.

## 도구 사용 규칙
1. `path`는 실제 수정할 파일의 정확한 경로를 사용하세요.
2. `edit_file.edits`는 반드시 배열 형태로 작성하세요.
   - 예시:
     ```json
     [
       {
         "oldText": "기존 코드",
         "newText": "수정 코드"
       }
     ]
     ```
3. `oldText`는 파일에 실제 존재하는 충분히 긴 문맥으로 지정하세요.
4. 의도하지 않은 대량 치환을 피하기 위해 최소 변경만 수행하세요.
5. 마지막 `read_file` 결과로 변경 반영 여부를 확인하세요.
